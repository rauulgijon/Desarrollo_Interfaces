
# FinalActivity.Wpf_MVC — estructura MVC

- **Models/**: clases de datos (DTOs), p.ej. `ApiObject`
- **Controller/**: acceso a la API (`ObjectController`)
- **View/**: XAML y code-behind (`MainWindow.xaml`) — `StartupUri="View/MainWindow.xaml"`

## Requisitos
- Windows + .NET 6 SDK
- Visual Studio 2022 o posterior

## Ejecución
1. Abrir `FinalActivity.Wpf_MVC.csproj` en Visual Studio.
2. Pulsar **F5**.
3. Clic en **Cargar datos**.
